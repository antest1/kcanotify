[Common] Choose which app you are use to play KanColle. Turn on button(s) on top and run Kancolle via GAME START button.
[GotoBrowser] Set sniffer mode as PASSIVE and activate Broadcast Mode on GotoBrowser.
[OfficialApp] Set sniffer mode as ACTIVE (this is default setting)

Main Features
- Expedition/Docking/Morale Notifications
- BattleView / QuestView
- Node tracking during sorties
- Heavy Damage Alert (prior to/during combat)
- LoS/FP/Fatigue indicators
- Tools like Ship/Equipment List, Improvement Arsenal Chart
